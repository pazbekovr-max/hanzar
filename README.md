# Random Quote Generator (Генератор случайных цитат)

## Автор
Ханзярова Самира Рустамовна

## Описание
Это графическое приложение на Python, использующее библиотеку Tkinter. Оно позволяет генерировать случайные цитаты из предопределенного списка, добавлять новые, фильтровать их по автору и теме, а также сохранять историю генераций в файл JSON.

## Возможности
- Случайная генерация цитат.
- Сохранение и загрузка истории из JSON.
- Фильтрация по автору и теме.
- Валидация ввода (защита от пустых полей).
- Простой и интуитивный GUI.

## Установка и запуск
1. Убедитесь, что у вас установлен Python 3.x.
2. Клонируйте репозиторий.
3. Запустите файл `main.py`:
   ```bash
   python main.py
